Welcome to HEPCon's documentation!
===================================

**HEPCon**  is the cross mobile application for HEP events
that integrates with Indico and helps organizers and attendees to interact each other with ease.

It pulls data from an Indico page and offers a *simple* and *intuitive* UI.

For the conference organizers HEPCon helps to:
1. Create announcements

For the conference conveners HEPCon helps to:
1. Start and stop presentations dynamically
2. Receive questions from attendees to ask
3. Receive requests from attendees to ask questions themselves

For the attendees HEPCon helps to:
1. Create reminders for specific presentations
2. Favorite presentations
3. Rate presentations
4. List, filter, group presentations

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
