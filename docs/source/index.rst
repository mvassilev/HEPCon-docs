Welcome to HEPCon's documentation!
===================================

**HEPCon**  is the cross mobile application for HEP events
that integrates with Indico and helps organizers and attendees to interact each other with ease.

It pulls data from an Indico page and offers additional functionality to help organizers and attendees better interact with each other.

Check out the :doc:`usage` section for further information.

.. note::

   This project is under active development.
